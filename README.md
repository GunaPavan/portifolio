# Gunaputra Nagendra Pavan Yedida — Portfolio

Personal portfolio website showcasing my work as an AI/ML Engineer, Researcher, and Co-Founder of Discensys.

## Live Site

[gunapavan.github.io/portifolio](https://gunapavan.github.io/portifolio/)

## Sections

- **About** — Background, education, and research interests
- **Experience** — Timeline of roles across industry, research, and open-source
- **Projects** — 12 projects linked to GitHub repositories
- **Research** — 6 publications linked to Zenodo
- **Skills & Certifications** — Technical skills, 10 certifications, and achievements
- **Volunteering** — Zooniverse and Smithsonian Institution contributions

## Features

- Dark theme with interactive neural network background
- Animated CSS robot mascot
- Custom robot cursor follower with glow trail
- 3D tilt effect on project cards with glare
- Magnetic buttons and floating skill tags
- Typed text animation cycling through roles
- Fully responsive design

## Tech Stack

HTML | CSS | JavaScript — No frameworks, no dependencies, pure vanilla.
